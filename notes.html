Tags I learned

<h1> ... <h6>
Headings that organize the page content from most important to least important.
<p>
Defines a paragraph of text.
<a>
Creates a link to another page or resource using the href attribute.
<img>
Displays an image using src and provides accessibility text with alt.
<nav>
Defines a section of navigation links for the website.
